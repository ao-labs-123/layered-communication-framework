## 5. Multi-label Policy
**Layer**

Layerは原則単一ラベル。

最も支配的な層を選択する。

**Subject**

Subjectは原則単一ラベル。

発話の態度が最も向いている対象を選択する。

**Stance**

Stanceのみ複数ラベルを許可する。

**許可例**
```stance
批判/要求
共感/賞賛
不安/要求
皮肉/批判
```
**非推奨**
```stance
賞賛/批判
共感/皮肉
```
